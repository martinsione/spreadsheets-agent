# Support

## Getting Help

OpenSheets is open source software maintained by the community.

### GitHub Issues

The best way to get help is to open an issue on GitHub:

**[github.com/martinsione/opensheets/issues](https://github.com/martinsione/opensheets/issues)**

Before opening a new issue, please:

1. Search existing issues to see if your question has been answered
2. Include your browser, spreadsheet platform (Google Sheets/Excel), and steps to reproduce

### Feature Requests

Have an idea? Open a GitHub issue with the "enhancement" label or submit a pull request!

### Bug Reports

Found a bug? Please open a GitHub issue with:

- What you expected to happen
- What actually happened
- Steps to reproduce
- Browser and platform details

## No Guaranteed Support

As a free, open source project, support is provided on a best-effort basis by the community. There are no guaranteed response times or SLAs.
